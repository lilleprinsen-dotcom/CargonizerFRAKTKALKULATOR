# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.1.0] - 2026-04-07
### Added
- Test pyramid with unit, integration, and checkout smoke (E2E) suites.
- CI workflow for PHPCS, PHPStan, Psalm, and PHPUnit matrix execution.
- Rollback-safe migration manager with idempotent settings defaults.
- Rollout checklist for staging soak, load testing, and monitored production rollout.

### Changed
- Version metadata bumped to 2.1.0.

## [2.0.0] - 2026-04-01
### Added
- Modular architecture and compatibility adapters.
